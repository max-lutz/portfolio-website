French national assembly data visualization

Data visualization of freely available data from the French national assembly during the year 2017 and 2022.

Online app Github

This dashboard was developed to visualize the data from the French national assembly. The user can use filters to switch between pages to display different aspects of the datasets. 

The data was preprocessed using pandas and python and the graphs were created using matplotlib and seaborn.
