Flood map comparison

Comparing historical flood maps with actual flood maps in St Louis.

Article

Description

I did an analysis of historical flood maps versus actual flood maps for a series of post on social media for the company I was working for, Tenko. This analysis highlighted the fact the historical flood maps are becoming outdated.

I was part of a team that implemented a script to download data and generate aggregated flood maps based on coordinates and timestamps. Historical data from FEMA and recent flood data was compiled to created this data visualization. 

The code source is property of Tenko.
