High dimensional dataset visualization

Data visualization application to display datasets with hundreds of feature on a 2D graph.

Online app Github

Description

Online application to visualize a dataset with a lot of features. The user can upload his own dataset. It will be preprocessed automatically and reduced to two features which can then be displayed on a 2D graph. Basic datasets are also available to choose from.

The application uses python, scikit-learn, seaborn and matplotlib.
