Classification pipeline tool

Application to generate preprocessing and classification pipeline in a few seconds.

Online app
 Github

Description

I developed an application to help people test and generated pipelines for classification problems. The user can apply a combination of thousands of preprocessing steps, select a model and its hyperparameters and get the performance of his model.

This app is fully responsive and retrains a model whenever a change is detected in the pipeline. Finally, you can download the pipeline and reuse it in your own projects.

The user can choose from different basic datasets already available or upload his own dataset. The app was developped using python, pandas, scikit-learn and streamlit.
